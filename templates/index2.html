About

The core of the project is a deep learning model that combines the image features with the text data. This model is an RNN (Recurrent Neural Network) and LSTM (Long Short-Term Memory) architecture.

  • To extract features from images CNN (Convolutional Neural Network) VGG16 architecture is used .
  • The model is trained to minimize a loss function, using categorical cross-entropy.
  • A tokenizer is used to create a vocabulary of words and map words to unique integer IDs.

We use evaluation metrics like BLEU (Bilingual Evaluation Understudy) to assess the quality of generated captions. BLEU measures how well the generated captions match human-written captions.
